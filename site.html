Implementing QAOA (Quantum Approximate Optimization Algorithm) to solve Vehicle Routing Problems (VRP) on NISQ devices.

Abstract

Classical combinatorial optimization faces exponential complexity scaling. We propose a hybrid variational approach using QAOA to approximate solutions for the Vehicle Routing Problem. By encoding city nodes into a Hamiltonian, we demonstrate a quadratic speedup in convergence compared to classical Simulated Annealing on sparse graphs.

The Hamiltonian

We map the VRP to an Ising model where the cost function is defined as:

$$ H_C = \sum_{i,j} w_{ij} Z_i Z_j + \lambda \sum_i (1-Z_i) $$
  • w_ij: Distance between nodes
  • Z_i: Pauli-Z operator (Spin)
  • λ: Penalty for unvisited nodes

Equal superposition created via Hadamard gates applied to all qubits initialized at |0⟩.

Cost Layer (Mixer)

Entanglement introduced via CNOT gates followed by parameterized Rz rotations (β, γ).
